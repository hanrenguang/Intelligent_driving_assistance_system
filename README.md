# 基于计算机视觉的智能驾驶辅助系统
智能驾驶辅助系统（课设）

## 运行环境
- Ubuntu16.04  
- CMake  
- OpenCV3.4  
- Python2.x/3.x

## 说明
本项目仅学习使用，且并未包含嵌软部分代码，运行时需修改 `src/autofollow` 下的源码，把PC与主控板通信部分代码注释掉

## 运行

首先进入 `lpr` 目录，运行车牌内容识别代码：
```bash
$ python server.py
```

然后回到根目录，执行以下命令并运行车辆跟随代码：
```bash
$ mkdir build
$ cd build
$ cmake ..
$ make -j6
$ ./Intelligent_driving
```
